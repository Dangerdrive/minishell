# Git

Switch to an existing branch
```bash
git switch <branch_name>
```

Create a new branch and switch to it
```bash
git switch -c <branch_name>
```

Get a specific file/folder from another branch
```bash
git checkout <origin_branch> <path/to/folder>
```
